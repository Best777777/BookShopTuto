# BookShopTuto - 书店管理系统数据库项目

## 项目简介

BookShopTuto是一个基于C# WinForms开发的书店管理系统数据库项目。该项目包含了完整的数据库设计和管理功能，适用于中小型书店的日常运营管理，包括图书管理、用户订单处理、管理员权限控制等功能。

## 技术栈

- **开发语言**: C#
- **框架**: .NET WinForms
- **数据库**: SQL Server
- **UI组件库**: Guna.UI2.WinForms 2.0.3.5
- **开发工具**: Visual Studio

## 主要功能

1. **图书管理**: 添加、编辑、删除和查询图书信息
2. **用户管理**: 管理客户信息，包括注册和基本信息维护
3. **订单处理**: 创建订单、查询订单状态
4. **管理员系统**: 基于角色的权限控制
5. **部门管理**: 管理书店内部部门信息

## 数据库设计

项目采用了关系型数据库设计，主要实体包括：

- **用户(User)**: 存储客户信息，如用户ID、姓名、电话等
- **书籍(Book)**: 存储图书信息，包括图书ID、标题、作者、价格等
- **管理员(Administrator)**: 存储管理员账户信息
- **订单(Bill)**: 记录销售订单数据
- **部门(Department)**: 管理书店内部部门结构

## 安装与使用

### 前置要求

- Visual Studio 2022或更高版本
- .NET Framework 4.x或更高版本
- SQL Server Express或更高版本

### 安装步骤

1. 克隆或下载本仓库到本地
2. 使用Visual Studio打开`BookShopTuto.sln`解决方案文件
3. 还原NuGet包依赖（右键解决方案 -> 还原NuGet包）
4. 附加`2331051304.mdf`数据库文件到SQL Server
5. 修改数据库连接字符串（如有必要）
6. 编译并运行项目

### 默认登录信息

- **管理员账户**: 请参考`管理员密码.txt`文件

## 注意事项

1. 本项目为教学演示用途，实际部署时请修改默认密码
2. 数据库文件可能需要根据您的SQL Server版本进行兼容性调整
3. 建议在开发环境中使用，生产环境请进行适当的安全加固
